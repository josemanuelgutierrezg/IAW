HTML5

TML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado HTML) para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros…

  • Sirven para organizar y estructurar una web.
  • Permiten definir jerarquías o niveles de orden.
  • Son compatibles con todo tipo de dispositivos.
  • Incluyen semántica pensada para la accesibilidad web.
  • Es posible anidarlos para construir contenido complejo.
